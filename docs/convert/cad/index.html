To CAD

This function allows the user to convert created body to 3D geometry in form of BRep files.

Usage​

Geometry Type​

In this case, Geometry type refers to the type of geometry to select

STL​

Converts an STL body, generated using the LDS plugin to a set of spheres and cylinders in a BRep format

Sketch-Lines​

Allows the user to select lattice structures in the 'sketch-line' format to be converted to a set of spheres and cylinders in a BRep format.

When converting from a 'sketch-line' format, two new options appear:

Single nodes​

This selection allows the user to convert 'single' nodes that are not connected (default behavior only creates spheres at the end of evert cylinder)

Filter out construction lines​

This checkbox is meant for user that desire to generate lattice structures out of their own 3D sketches. Ticking this box will only generate as cylinders and spheres sketch-lines that are not constructions.

Remark: When using sketch-lines generated using the LDS plugin, ticking this box will prevent anything from being generated as construction lines are used

Body​

Bodies to select. Either mesh bodies in the STL format or a set of sketch-lines.

Wire radius​

This parameter defines the radius of the cylinders being generated

Node Radius​

This parameter defines the radius of the spheres being generated
