Insert

This function allows the user to add a fully created screw/bolt insertion point based on a set of parameters

Usage​

Profile​

Sketch profile to be used to generate an insert from. Only one can be selected at once

Insert Diameter/Length​

The diameter of the cylinder/screw/bolt that will be inserted dictates the size of the hole to be inserted. Screw length defines the extrusion length

Insert Type​

There are two main insert types, both corresponding to different use cases.

Square insertTriangle insert

Manual coordinates​

This checkbox allows to select wether or not the user wants to place the center of the hole manually. If left unchecked, the whole will automatically be put in the middle

X/Y/Z fields​

This field allows to put the xyz coordinates for the hole (in mm and cartesian reference frame)

Direction​

This defines the direction to extrude in. Outwards will mean in the same direction as the normal of the profile an inwards will be the opposite
