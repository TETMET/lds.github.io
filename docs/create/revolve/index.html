Revolve

This function allows the user to create geometry based off of a rotational sweep around an axis

Usage​

Profiles​

The selected profiles that will be rotated around the axis to create the 3D Lattice structure

Axis​

The axis around which the profiles will be swept

Structure type​

This refers to the unit cell structure that will be used to create the revolved geometry. For more information on lattice generation, read more here in the 'unit-cell' section

Revolve Rotation​

This refers to the amount (in degrees) of rotation the revolve function will do (i.e: the 'portion' of the circle)

Revolve steps​

This refers to the number of repetitions the profile will be drawn along the revolution. This can be scene as a 'resolution of sorts'

6 Steps Revolve12 Steps Revolve

Remarks​

The revolve function can easily create sharp angles along the revolution which can be 'impossible' to build for some production processes. We recommend you run the repair function to prevent those issues
